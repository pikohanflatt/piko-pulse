# 開発ログ 2026-02-20 #001

## 実行したこと
- `backend/` の初期化: `npm init -y` → `typescript`, `tsx` をインストール → `tsconfig.json` 生成
- `backend/src/index.ts` に Hello World を書いて `npx tsx` で実行確認
- `frontend/` の初期化: `npm create vite@latest` で React + TypeScript プロジェクトを生成、開発サーバーの起動を確認
- backend に ESLint を導入: `npm init @eslint/config@latest` で対話形式セットアップ
- backend に Prettier を導入: `prettier` + `eslint-config-prettier` をインストール、`.prettierrc` 作成、`eslint.config.mjs` に連携設定を追加
- Prettier の動作確認: `--check` で差分検出 → `--write` で自動修正を体験

## 詰まったところ
- ESLint セットアップの選択肢で「TypeScript」が直接なく迷ったが、「JavaScript」を選んだあとに TypeScript の質問が出た
- `eslint.config.mjs` の `globals.browser` が `globals.node` になっているべきだった（セットアップ時の設定が反映されていなかった）
